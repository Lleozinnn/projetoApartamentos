# Projeto Apartamentos

Sistema desenvolvido como Trabalho Interdisciplinar do **1º período** do curso de **Tecnologia em Sistemas para Internet** do **IFES Campus Santa Teresa**.

O objetivo do projeto foi aplicar, de forma integrada, os conhecimentos adquiridos ao longo do semestre por meio do desenvolvimento de uma aplicação web para gerenciamento de apartamentos destinados à venda e aluguel.

## Sobre o projeto

O Projeto Apartamentos é uma aplicação web que permite o cadastro, gerenciamento e consulta de apartamentos. O sistema possui funcionalidades voltadas tanto para usuários quanto para administradores, proporcionando uma interface simples, organizada e responsiva.

Este projeto foi desenvolvido para colocar em prática conceitos fundamentais de desenvolvimento web, manipulação de dados e trabalho em equipe utilizando controle de versão.

## Funcionalidades

* Cadastro de apartamentos;
* Listagem de imóveis;
* Visualização dos detalhes dos apartamentos;
* Cadastro de produtos do estoque;
* Controle de acesso para administradores;
* Pesquisa e filtragem de imóveis;
* Interface responsiva;
* Armazenamento de dados utilizando Local Storage.

Clone o repositório:

```bash id="o9q6z1"
git clone https://github.com/Lleozinnn/projetoApartamentos.git
```

Acesse a pasta do projeto e abra o arquivo `index.html` em um navegador.

> O projeto não necessita de instalação de dependências ou servidor, pois foi desenvolvido utilizando tecnologias web nativas.
